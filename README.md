# Correr el proyecto en Desarrollo

1. Clonar el repo
2. Crear una copia de `.env.template` a `.env` y cambiar las variables según tu configuración
3. Instalar las dependencias con el comando

```bash
npm install
```

4. Crear una base de datos con el comando

```bash
docker compose up -d
```

5. Ejecutar el proyecto en modo desarrollo con el comando

```bash
npm run dev
```
